# Customer Logos

Place company logos in this folder with the following filenames:

1. **nomad-tanzania.png** - Nomad Tanzania Limited
2. **mwalimu-nyerere-foundation.png** - Mwalimu Nyerere Foundation
3. **tazama-pipeline.png** - TAZAMA Pipeline
4. **cotwu-tanzania.png** - COTWU Tanzania
5. **tanzania-institute-project-management.png** - Tanzania Institute of Project Management

## Logo Requirements:
- Format: PNG (preferred) or JPG
- Recommended size: 200-300px width
- Background: Transparent or white
- Aspect ratio: Maintain original logo proportions

## How to Add Logos:
1. Download logos from company websites (with permission)
2. Save them with the exact filenames listed above
3. Place them in this folder (`public/images/customers/`)
4. The website will automatically display them

If a logo file doesn't exist, the company name will be displayed as text instead.
