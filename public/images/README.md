# Images Directory

This folder contains all image assets for the HRUNLOCK Africa website.

## Folder Structure

- **gallery/** - Gallery images for events and activities
- **customers/** - Customer/partner logos
- **team/** - Team member photos
- **logo/** - Brand logos and icons
- **hero/** - Hero section images/backgrounds

## Image Guidelines

- Use optimized images (WebP, JPEG, or PNG)
- Recommended dimensions:
  - Gallery: 1200x800px (4:3 aspect ratio)
  - Customer logos: 300x200px (flexible)
  - Team photos: 400x400px (square)
  - Hero images: 1920x1080px (16:9 aspect ratio)
- Keep file sizes reasonable for web performance
- Use descriptive filenames (e.g., `team-john-doe.jpg`)

## Upload Your Images

Place your images in the appropriate subfolders based on their purpose.
